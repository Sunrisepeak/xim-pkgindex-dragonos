# xim-pkgindex-dragonos

xim package index for dragonos community

## 添加索引仓库

> 添加索引仓库到xim的索引仓库管理器中

```bash
xim --add-indexrepo dragonos:https://github.com/sunrisepeak/xim-pkgindex-dragonos.git
xim --update index
```

## 使用仓库中的包

```bash
xlings install dragonos:dragonos-dev
```
